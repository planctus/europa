1Datepicker #permalink

This example is based on jQuery Datepicker widget. It requires both jQuery and jQuery UI libraries.
Also, it assumes that the default CSS styles from jQuery UI are included as well.
The CSS applied from styleguide is an override over the default CSS of jQuery UI.

Example 
<script src="public/js/components/datepicker/jquery.ui.datepicker.min.js"></script>
<script src="public/js/components/datepicker/datepicker.js"></script>
<link rel="stylesheet" href="public/css/jquery.ui.core.min.css">
<link rel="stylesheet" href="public/css/jquery.ui.datepicker.min.css">

<label for="datepicker">Date:</label>
<input class="form-control" type="text" id="datepicker">
