1Datepicker #permalink

This example is based on jQuery Datepicker widget. It requires both jQuery and jQuery UI libraries. Also, it assumes that the default CSS styles from jQuery UI are included as well. The CSS applied from styleguide is an override over the default CSS of jQuery UI.

Example

Date: 

<link rel="stylesheet" href="sass/vendor/css/jquery.ui.core.min.css">
<link rel="stylesheet" href="sass/vendor/css/jquery.ui.datepicker.min.css">
<p>Date: <input type="text" id="datepicker"></p>
