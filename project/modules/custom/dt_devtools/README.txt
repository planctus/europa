Module: dt_devtools

Description
===========
It will load locally Web tools load.js file. This is a good case when you are working outside
of the commission network.

Requirements
============
* None.

Installation
============
Like any other module.

Usage
=====
In the settings page (admin/config/development/dt_devtools), select Web tools (load.js)
Create your Webtools bean and it should be working.

Notes
=====
Avoid committing this module to production.
